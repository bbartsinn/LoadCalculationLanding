# Real World Electric Load Calculator Landing Page

Static SEO landing page for `smartplanning.realworldelectric.com`.

Primary funnel:

1. Rank for Calgary electrical load calculation searches.
2. Send users to `https://loadcalculation.realworldelectric.com/`.
3. Convert review/call/WhatsApp requests for Real World Electric.

## Local Preview

```powershell
python -m http.server 4173
```

Open `http://127.0.0.1:4173`.

## Phase 1 Included

- Search-focused title and description
- Canonical URL
- OpenGraph/Twitter metadata
- LocalBusiness, SoftwareApplication, and FAQ structured data
- `robots.txt`
- `sitemap.xml`
- Mobile responsive landing page
